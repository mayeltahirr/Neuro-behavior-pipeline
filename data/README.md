# Example data (synthetic)

Everything in this folder is fake, generated by `src/simulate.py` -- not
recordings from a real animal. I'm including a small example of it here so
someone can see the shape of the data without having to run the pipeline
first.

- **`example_behavior_bouts.csv`** -- a 2-minute synthetic ethogram: which
  behavior (rest, groom, or move) was "happening" during each bout.
- **`example_spike_times.csv`** -- synthetic spike timestamps for 6 fake
  neurons over that same 2 minutes, generated so firing rate depends on
  behavior state.

Regenerate these (or make a bigger/smaller version) with:

```bash
python -c "
from src.simulate import generate_behavior_bouts, generate_spike_trains
bouts = generate_behavior_bouts(duration_s=120, seed=42)
spikes = generate_spike_trains(bouts, n_neurons=6, seed=42)
"
```

The full pipeline (`src/pipeline.py`) generates its own data on the fly at
whatever size you ask for -- these two files are just a fixed, small
snapshot for quick browsing.
